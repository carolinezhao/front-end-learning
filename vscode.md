调试代码：view -> Integrated Terminal

shortcut key
* 注释光标所在整行: cmd ＋ "/"
* run: ctrl + option + N (powered by Code Runner)
* word wrap: option + z

auto-completion `tab`
* 生成基本页面框架 html:5
* 多个相同元素 div*3
* 元素及常用属性 img

Extension
* Code Runner
* ESLint
* Auto Close Tag (for HTML)
* theme

在 VS Code 中运行 js
* 插件 Code Runner 指定 js 的运行环境为 node (系统已安装 node)
* 查看版本及路径：
```js
console.log(process.versions);
console.log(process.argv);
```
