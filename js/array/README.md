## 相关数据结构

**栈**：LIFO (Last-In-First-Out，后进先出)

    length = array.push('item') 后边进
    item = array.pop() 后边出

**队列**：FIFO (First-In-First-Out，先进先出)

    length = array.push('item') 后边进
    item = array.shift() 前边出

    length = array.unshift('item') 前边进
    item = array.pop() 后边出
