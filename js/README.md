- this --> function/this
- 原型链和继承 --> object
- 异步 --> async

## 调试

- 浏览器 DevTool

## HTML 中的 js

最佳实践：关闭 `<body>` 前通过 `<script>` 引入 js 文件。<br>
浏览器会在加载脚本之前解析和显示 HTML，有利于提升页面性能。
