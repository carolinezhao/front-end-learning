图片无 float 效果。

float 的设计初衷是用于文字环绕效果。用于网页布局时会带来以下影响：

特点1 脱离文档流。破坏父元素结构 (塌缩)，影响与父元素同级的元素。

使用了 float 的文本。

特点2 包裹性。

宽度会自动调整为包裹住内容的宽度 (从而体现环绕效果)，而不是撑满整个父容器。

但要注意 display 样式没有变化，仍是 display: block

特点3 清空格。

正常的 img 中间是有空格的，因为多个 img 标签之间的换行会被浏览器识别为空格。

这一特性的根本原因是 float 会导致节点脱离文档流结构，因此它们就尽量往一边靠拢。

清除 float 的影响 (仅针对脱离文档流)

使用了 float 的文本。

方法1 浮动元素的父标签使用 after 伪对象 (见 CSS)。

方法2 浮动元素的父标签设置 overflow 属性为 auto 或 hidden。
