## 1.3 网络基础 TCP/IP

TCP/IP 协议族：是互联网相关的各类协议的总称。HTTP 是其内部的一个子集。

### 分层管理

* 应用层：FTP，DNS，HTTP
* 传输层：TCP，UDP
* 网络层：IP
* 链路层：硬件

### 通信传输流

发送端 (每通过一层增加首部)：应用层 --> 传输层 --> 网络层 --> 链路层  ---| <br>
接收端 (每通过一层删除首部)：应用层 <-- 传输层 <-- 网络层 <-- 链路层  <--| <br>

### IP 协议：负责传输

### TCP 协议：确保可靠性

### DNS 服务：负责域名解析
